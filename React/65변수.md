const - 한 번 지정하고나면 변경 불가능한 상수 선언
let - 동적인 값을 담을 수 있는 변수 선언
var - 새로 선언하면 변경된 값이 나타남
<br>
let은 한번 선언한 후 값이 유동적으로 변할 수 있을 때만 사용(ex: for)
const는 한번 설정한 후 변할 일이 없는 값에 사용



```javascript
function myFunction() {
	var a = "hello";
	if(true) {
		var a = 'bye';
		console.log(a); // bye
	}
	console.log(a); // bye
}
myFunction();
```



```javascript
function myFunction() {
    let a = 1;
    if(true) {
        let a = 2;
        console.log(a); // 2
    }
    console.log(a); // 1
}
myFunction();
```
